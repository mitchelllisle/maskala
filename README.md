# Maskala 

### KAnonymity

This library provides utility functions to assess and ensure K-Anonymity on datasets using Apache Spark.

#### Features

- Compute the K-Anonymity of a given dataset.
- Filter rows in a dataset to ensure K-Anonymity.

## Getting Started

### Dependencies

- Apache Spark 3.x
